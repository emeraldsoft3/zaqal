# Phase 10: Verification & SoC Integration

The final step is proving it works and making it a real chip.

## The Verification Loop
- [ ] **Difftest**: Running your Zaqal side-by-side with a golden model (NEMU or Spike).
- [ ] **Random Instruction Generation (Torture)**: Constant stress testing.
- [ ] **UVM (Universal Verification Methodology)**: Industry-standard testing.

## SoC Integration
- [ ] **AXI4 Bus Interface**: Connecting to DDR4 memory and Peripherals.
- [ ] **Booting Linux**: The ultimate test of correctness.
- [ ] **Benchmarks**: Running CoreMark, SpecInt, and Dhrystone to measure performance.
